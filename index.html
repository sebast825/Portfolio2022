Medical Appointment Management API

The robust and scalable solution for efficient medical appointment management

The Challenge

Traditional medical appointment systems are slow, insecure, and fail to scale. A robust solution was needed to guarantee confidentiality and the efficient handling of thousands of daily appointments.

The Solution

I designed and developed a REST API with a layered architecture (Repository Pattern) in .NET 8, implementing JWT for medical-grade security and an optimized algorithm to check availability in milliseconds.

Achieved Impact

  • Data protection through JWT and encryption.
  • Availability checking 15x faster than the previous system.
  • Foundation for 3 applications (web, mobile, admin) thanks to its modular design.
